# Modern JavaScript Exercises

A small browser project for practicing modern JavaScript basics and common mini tasks.

## How to use

Open `index.html` in a browser.

Work through the lessons from top to bottom:

1. Read the goal.
2. Follow the steps.
3. Copy the starter code into your browser console or a separate `.js` file.
4. Use the interactive demo to see the idea in action.
5. Mark the lesson as done.

Your progress is saved in the browser with `localStorage`.

## Topics included

- Variables
- Functions
- Arrow Functions
- Template Literals
- Objects
- Arrays
- `map()`
- `filter()`
- `reduce()`
- Destructuring
- Spread Operator
- Promise
- `async/await`
- Fetch API
